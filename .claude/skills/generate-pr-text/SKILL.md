# generate-pr-text

現在のブランチの変更内容からPR記載用の文章を生成するスキル。

## Usage

```
/generate-pr-text
```

## Behavior

1. `git log` と `git diff` からベースブランチとの差分を取得する
2. 変更内容を分析し、施策単位でグルーピングする
3. `.github/pull_request_template.md` のフォーマットに従ってPR文章を生成する
4. 生成結果はテキストとして出力する（ファイルには書き込まない）

## Rules

- 内容は日本語で記述する
- 「主な変更点」は施策タイトルを親リスト (`-`)、詳細を子リスト (`  -`) のネスト形式で記述する。見出し (`###`) は使わない
- 概要は1〜2行で簡潔にまとめる
- 破壊的変更がない場合は「なし」と記載する
- テストセクションには変更内容に応じたチェック項目を提案する
